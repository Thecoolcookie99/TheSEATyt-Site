TI-Nspire CAS Graphing Calculator 3D engine (AI)

The TI-Nspire CAS Graphing Calculator 3D engine is a project I worked on to create a 3D rendering engine for the TI-Nspire CAS calculator. I used AI to generate almost all of the code.

About The TI-Nspire CAS Graphing Calculator 3D engine - I used AI to create a polygon rendering engine for the TI-Nspire CAS calculator. It is the first (that I know of) 3D engine for the TI-Nspire. For some reason, the movement controls only work in the emulator although the actual rendering works fine on the calculator.

Download link: Download the Beta Version

How to run: Download the TNS file from the link above, then open it in TI-Nspire CAS software or upload it to the calculator using the online file transfer application (requires chrome).
