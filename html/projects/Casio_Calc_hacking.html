This page outlines how we are currently dumping the casio calculator: the FX82-AU plus II 2nd edition

About the project

The FX82-AU plus II 2nd edition that is the current school calculator for Australian studens in middle/upper school. This is how we obtained seg0 of the calculator's ROM (read-only memory). To start off, we need to overflow the calculator. This is neccassary for the future steps in which we craft a hackstring to display the ROM on the screen. To overflow, we first need to obtain an unstable character. This character has specific properties
